A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.
This project demonstrates DOM manipulation, event handling, and basic game logic.

🚀 Features
1 🎲 Random choice generation for both players
2 🖼️ Dynamic image rendering
3 🧠 Correct game logic (Rock > Scissor, Scissor > Paper, Paper > Rock)
4 🔄 Reset functionality
5 🎨 Clean and responsive UI

🛠️ Tech Stack:-
  1 HTML
  2 CSS
  3 JavaScript

📂 Project Structure:-
  Rock_paper_scissor/
  │── index.html
  │── style.css
  │── first.js
  │── image/
  │     ├── rock.jpg
  │     ├── paper.png
  │     └── scissor.jpg

▶️ How to Run

1. Clone the repository:
   
   git clone https://github.com/your-username/rock-paper-scissor.git

2. Open the project folder:

   cd rock-paper-scissor
   
3. Run the project:

   -> Open "index.html" in your browser

🎯 Game Rules:-
  1 Rock beats Scissor
  2 Scissor beats Paper
  3 Paper beats Rock
  4 Same choices result in a Draw
